# homebrew-audiobook-studio

Homebrew tap for [Audiobook Studio](https://github.com/StevenJPx2/audiobook-studio) —
turn a book PDF into a chaptered `.m4b` audiobook with a local LLM + Kokoro TTS.

macOS / Apple Silicon only.

## Install

```sh
brew install StevenJPx2/audiobook-studio/audiobook-studio
abs doctor
```

The formula's `url`/`sha256` are updated automatically by the app repo's
release workflow on each `v*` tag. Do not hand-edit them.
